# Company Pilot Runbook

1. Obtain written data-use permission and retention policy.
2. Select exactly one pattern/SKU, one camera, and one lighting condition.
3. Capture 5–10 normal references with stable acquisition metadata.
4. Seal 20–40 blind review images and collect expert labels; masks are preferred.
5. Import with source/capture-session identity and run duplicate/leakage audit.
6. Calibrate a new company threshold on validation data only.
7. Evaluate the sealed pilot set and report raw FP/FN counts, recall, confidence intervals,
   latency, and representative failures.
8. Lock a separate company threshold only after expert approval. Do not generalize results to
   other patterns, cameras, lighting, or factory conditions.
